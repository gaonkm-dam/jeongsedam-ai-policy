# 정세담 정책 프로그램

정책의 기획, 실행, 홍보, 성과관리까지 전 과정을 자동화하는 AI 기반 정책 시스템

## 주요 기능

### ✨ 핵심 개선 사항

1. **즉시 생성**: 버튼 클릭 즉시 이미지/영상 프롬프트 생성 및 화면 표시
2. **새로고침 기능**: 클릭마다 새로운 컨텐츠 자동 생성
3. **정책 프로세스 워크플로우**: 기획 → 실행 → 홍보 → 성과관리 단계별 진행
4. **대상별 맞춤 컨텐츠**: 시민/청년/노인/학부모 등 타겟별 자동 생성
5. **데이터 축적 및 분석**: 모든 정책 데이터 저장 및 성과 추적

### 🎯 주요 모듈

#### 1. 정책 기획 (Policy Planning)
- AI 기반 정책 목표 및 전략 수립
- 대상 분석 및 니즈 파악
- 핵심 전략 및 기대 효과 도출

#### 2. 실행 계획 (Execution Plan)
- 단계별 실행 항목 생성
- 리스크 관리 및 완화 방안
- 필요 자원 및 예산 범위 제시

#### 3. 커뮤니케이션 전략 (Communication)
- 핵심 메시지 개발
- 대상별 맞춤 메시지 생성
- 채널별 콘텐츠 전략

#### 4. 콘텐츠 생성 (Content Generation)
- **이미지**: DALL-E 3 기반 고품질 이미지 즉시 생성
- **영상**: 상세한 영상 제작 프롬프트 자동 생성
- **문서**: PDF 보고서, ZIP 패키지 자동 생성

#### 5. 성과 관리 (Performance)
- KPI 프레임워크 자동 생성
- 측정 방법 및 데이터 출처 제시
- 성과 지표 추적 및 모니터링

## 설치 및 실행

### 1. 필수 요구사항

- Python 3.11
- OpenAI API 키

### 2. 설치

```bash
# 저장소 클론 또는 다운로드 후

# 가상환경 생성 (선택)
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Mac/Linux

# 패키지 설치
pip install -r requirements.txt
```

### 3. 환경 설정

`.env.example` 파일을 복사하여 `.env` 파일 생성:

```bash
copy .env.example .env  # Windows
# cp .env.example .env  # Mac/Linux
```

`.env` 파일을 열어 OpenAI API 키 입력:

```
OPENAI_API_KEY=sk-your-actual-api-key-here
```

### 4. 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501` 열림

## 사용 방법

### 1단계: 정책 입력

1. "📝 정책 입력" 탭에서 정책 정보 입력
   - 정책 제목
   - 카테고리 (환경, 안전, ESG, 교육 등)
   - 대상 (시민, 청년, 노인, 학부모 등)
   - 정책 설명 (상세할수록 좋음)
   - 강조 키워드
   - 제약 조건 (예산, 기간 등)

2. 콘텐츠 패키지 선택
   - A 마케팅: 이미지 + 영상 + 홍보
   - B 정책 설명: 정책 문서 + PPT + FAQ
   - C 풀 패키지: 모든 항목 포함

3. "🚀 AI 분석 생성" 클릭

### 2단계: AI 분석 검토

"🤖 AI 분석 생성" 탭에서 생성된 내용 확인:
- 정책 기획 (목표, 전략, 기대효과)
- 실행 계획 (실행 항목, 리스크 관리)
- 커뮤니케이션 전략 (메시지, 채널)
- 성과 지표 (KPI, 측정 방법)
- 콘텐츠 제작 브리프 (이미지/영상)
- 마케팅 자료 (슬로건, FAQ)

### 3단계: 이미지 생성

"🖼️ 이미지 생성" 탭에서:
1. 이미지 크기 선택 (1024x1024, 1024x1792, 1792x1024)
2. 품질 선택 (standard, hd)
3. "🖼️ 이미지 1 생성" 클릭 → 20-40초 후 이미지 표시
4. "🖼️ 이미지 2 생성" 클릭 → 다른 컨셉의 이미지 생성
5. "🔄 새로고침" 클릭 → 추가 이미지 2장 생성
6. 생성된 이미지는 즉시 화면에 표시되고 다운로드 가능

### 4단계: 영상 프롬프트

"🎬 영상 프롬프트" 탭에서:
1. 영상 길이 선택 (10초, 20초, 30초, 60초)
2. 제작 플랫폼 선택 (Sora, Runway, Pika, Luma)
3. "🎬 영상 프롬프트 생성" 클릭
4. 생성된 프롬프트 복사하여 영상 제작 플랫폼에 활용
5. "바로가기" 버튼으로 플랫폼 접속

### 5단계: 결과 다운로드

"📊 결과 및 내보내기" 탭에서:
1. **📄 PDF 보고서**: 전체 분석 내용을 한글 지원 PDF로 다운로드
2. **📦 전체 ZIP**: 모든 이미지, 프롬프트, 분석 데이터를 압축 다운로드
3. 정책 상태 변경 (draft → active → completed → archived)
4. 성과 지표 입력 (조회수, 참여도, 만족도)

## 프로젝트 구조

```
정세담 정책 프로그램/
├── app.py                 # 메인 애플리케이션
├── requirements.txt       # Python 패키지
├── runtime.txt           # Python 버전
├── .env.example          # 환경 변수 템플릿
├── config/
│   └── settings.py       # 전역 설정
├── modules/
│   ├── __init__.py
│   ├── database.py       # 데이터베이스 관리
│   ├── ai_engine.py      # AI 분석 엔진
│   ├── image_generator.py # 이미지 생성
│   └── export_manager.py  # PDF/ZIP 생성
├── data/
│   └── policies.db       # SQLite 데이터베이스 (자동 생성)
└── assets/               # 정적 파일
```

## 데이터베이스 스키마

### policies
- 정책 기본 정보 (제목, 카테고리, 대상, 설명, 상태)

### policy_contents
- 정책 관련 콘텐츠 (AI 분석 결과, 문서 등)

### generated_media
- 생성된 미디어 (이미지, 영상 프롬프트)

### policy_performance
- 정책 성과 데이터 (조회수, 참여도, 만족도 등)

## 기술 스택

- **Frontend/UI**: Streamlit
- **AI Engine**: OpenAI GPT-4, DALL-E 3
- **Database**: SQLite
- **Document**: ReportLab (PDF)
- **Image**: Pillow (PIL)

## 주요 개선 사항 (vs 기존 버전)

| 기능 | 기존 | 신규 |
|------|------|------|
| 이미지 생성 | 프롬프트만 제공 | 버튼 클릭 → 즉시 생성 및 표시 |
| 새로고침 | 동일 컨텐츠 반복 | 클릭마다 새로운 컨텐츠 |
| 프로세스 | 단계 구분 불명확 | 기획→실행→홍보→성과 워크플로우 |
| 대상별 메시지 | 수동 작성 | 7개 타겟별 자동 생성 |
| 데이터 관리 | 세션만 저장 | DB에 영구 저장 + 검색 |
| 영상 | 단순 프롬프트 | 타임라인, 장면, 오디오까지 상세 |
| 성과 관리 | 없음 | KPI 추적 및 모니터링 |
| 내보내기 | 기본 ZIP | 한글 PDF + 구조화된 ZIP |
